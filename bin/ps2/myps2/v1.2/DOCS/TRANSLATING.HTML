Translating myPS2

All language strings used by the menus are stored in an XML file.
The default myPS2 language XML file is located in the installation
directory under /language/english.xml

An entry in the file looks like this:

        <string>
            <id>3</id>
            <value>My Files</value>
        </string>

The id field is a unique number used to identify the string from the source code. You don't have
to care about it when translating the file. Just make a copy of the english.xml file, rename it and
translate all of the strings entries.

The language xml file must be saved as UTF-8 encoded! Most text editors let you choose
the encoding in the "save as" dialog.

Every myPS2 skin has atleast support for the Latin-1 charset.This charset covers most of the
western european languages (Go here for a complete list of the ISO-8859 Charsets).

Additionally a skin can supply fontsets for different character encodings, for instance the myPS2
default skin provides font files for modern greek and languages that use the cyrillic alphabet. If
you want to create your own set of font files for your translation, read the "Font Files" section in
the Skinning.html file.

When you are done with your translation, copy the translated file into the /language/ directory
and it will show up in the language selection menu in myPS2 where it can then be selected.
